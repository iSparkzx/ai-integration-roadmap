Phase Details

What Each Phase Delivers

Clear deliverables, owners, and exit criteria so nothing falls through the cracks.

Phase
00
Process Mapping & Baseline
Deadline: June 30th

Before introducing AI, you need to know how Marketing actually works today. This phase is about mapping the current state — not the aspirational state.

  • Document and chart current workflows, procedures, and handoffs across the department
  • Identify broken processes, bottlenecks, and improvement areas
  • Establish baseline performance metrics for pre-AI comparison
  • Create a shared language around "how we do things now"
Workflow Mapping Gap Analysis Baseline Metrics
Phase
01
Foundation & Literacy
July 3rd – July 17th

AI fails when people don't know how to use it. This phase builds the human layer — the skills, documentation, and ownership structure that make adoption stick.

  • Identify fast adopters for AI stack training and rollout
  • Run a two-week intensive covering AI tools, skills, connectors, folder structure, and prompting
  • Create a Loom video library of key functionalities for self-service learning
  • Using Phase 0 process maps, pinpoint exactly where AI can help
  • Assign specific people as adoption owners with authority and freed-up time to lead

04
Measure, Govern & Sustain
Ongoing

The final phase is not an endpoint — it is the operating model. Track outcomes against baseline. Consolidate overlapping tools. Define who maintains agents, updates workflow maps, and trains new hires.

  • Track outcomes against the Phase 0 baseline metrics
  • Consolidate overlapping tools and redundant systems
  • Define ownership for agent maintenance, workflow updates, and onboarding
  • Embed AI into standard practice so it becomes "how we work," not a separate initiative
